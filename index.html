Hiking Blog

This is a blog focusing on hiking related topics, it features user authentication, ability to make posts, and profile viewing for logged in users. The website's data is managed through SQL database to maintain persistent posts in case the server goes down. This also includes the login process for already registered users and when a new user registers. Users will be required to sign in to a google account through OAuth to create an account or to sign in to an already registered one. User can also manage their account through a deletion feature and username change feature.
